Looks good so far. Estimation follows. We analyze all data together block 1 + block 2.

The estimation is not straight-forward because we have so much data with considerable variability. If we use beta parametrized by mean proportion \(\mu\) and sample size \(N\) distribution to model trial variability we have the following problem. Some subjects show sharply peaked distributions i.e. \(N\) goes to some large number. This causes problems for the sampler. If we use gamma prior, it takes eternity to converge the high \(N\). If we use flat prior with wide range the remaining \(N\)s (which are small around 10) are difficult to find.

We can try truncated gaussian instead, but we get an opposite and more severe problem. We have some subjects with very flat distribution so that the variance of the gaussian is pulled to infinity and mean becomes practically unidentifiable (because the gaussian is flat in the region [0,1]).

Our current best solution is to use a constrained flat prior [0,1000] or [0,100] for \(N\). This prior doesn't allow \(N\) (the variable nmu below in particular) to reach its true posterior, but we think it is justified because a) higher values are implausible and b) the absolute difference in terms of the variance \(\sigma= \mu*(1-\mu)/(1+N)\) between say \(N=1000\) and some \(N>>1000\) is very small and should not influence the estimation of \(\mu\).

model = """
data {
    int<lower=0> N; // number of subjects
    int<lower=0> T; // number of trials
    real owt[N,T]; // proportion of time spent in wolfpack areas
    int sel[N,T];
}
parameters {
    real<lower=0,upper=1> mu[N];
    real<lower=0,upper=100> nu[N];
    real<lower=0,upper=1> mmu;
    real<lower=0,upper=1000> nmu;
}
model {
    nmu~pareto(0.1,1.5);
    nu~pareto(0.1,1.5);
    for (n in 1:N){
        mu[n]~beta(mmu*nmu,(1-mmu)*nmu);
        for (t in 1:T){
            if (sel[n,t]==1) owt[n,t]~beta(mu[n]*nu[n],(1-mu[n])*nu[n]);
        }
    }
}
"""

Next we estimate the relation between the manipulated displacement and the preference for wolfpack areas. We do this by estimating the slope of a regression line between the two variables with manipulated displacement as predictor. This is not straightforward because we do not use a model with normally distributed errors. Nevertheless, we can try to supply linear equation for the mean proportion \(\mu= \alpha_0 + \alpha_1 *d\), where \(d\) the is the predictor (manipulated dispalacement). The problem with this approach is that we can't ensure that the result for \(\mu\) will be in the range \([0,1]\). However, we can get the sampler going by constraining \(\alpha_0\) and \(\alpha_1\) to particular range such that the result of the addition will be mostly in the range \([0,1]\).

We start by fitting a regression model to each subject separately and then we resume with a hierarchical model.
